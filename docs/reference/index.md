### 快速上手

::: tip 提示

本框架是基于 vue3 + vite 搭建，组件是基本 ant design vue 二次封装

:::

### 安装依赖

```bash:no-line-numbers
npm install
```

### docs文档结构目录
```
├─ .vitepress
│  └─ config.js           # vitepress配置文件
├─ reference                  # .md文件
├─ public                 # 静态资源文件
├─ index.md               # 文档home页面
```
