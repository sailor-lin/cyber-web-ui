# g-pagination 分页

| 属性名          | 说明                 | 类型                 | 默认值                 |
| :-------------- | :------------------- | :------------------- | :------------------- |
| total             | 总数              | Number      | 0         |
| current(v-model)  | 当前页数           | Number     | 1         |
| pageSize(v-model) | 每页条数           | Number     | 10        |
| left(v-slot)      | 左插槽             |            |           |

::: tip 提示

查看更多属性请前往 https://www.antdv.com/components/pagination-cn#API

:::
