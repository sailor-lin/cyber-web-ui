# g-tag-card 标签卡片

### 属性

| 属性名          | 说明                 | 类型                 | 默认值                 |
| :-------------- | :------------------- | :------------------- | :------------------- |
| pattern         | 颜色模式           | "" \| "primary" \| "dark" \| "warning" \| "error" \| "gray" \| "classic"           |             |
| label         | label           | String           | label          |
| value         | value           | String           | value          |
| type         | 标签类型          | "label" \| "annotations"     | label    |


::: tip 提示

查看更多属性请前往 https://www.antdv.com/components/tag-cn#API

:::
