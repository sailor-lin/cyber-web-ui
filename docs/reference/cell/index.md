# g-cell 单元格布局

### 属性

| 属性名          | 说明                 | 类型                 | 默认值                 |
| :-------------- | :------------------- | :------------------- | :------------------- |
| icon             | 图标标识            | String                   |             |
| title            | 标题文案            | String \| v-slot:default  | 标题        |
| tips             | 提示文案            | String \| v-slot:tips     |            |
| to               | 点击标题要跳转的路径 | String                   |             |


::: tip 提示

1. g-cell 一般配合表格使用，icon为左侧图标，title为当前行名称或id，点击可跳转到详情页；

:::
