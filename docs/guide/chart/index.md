# g-chart 图表

### 属性

| 属性名          | 说明                 | 类型                 | 默认值                 |
| :-------------- | :------------------- | :------------------- | :------------------- |
| options         | 图表的配置信息        | echarts.options    |             |
| resize          | 是否开启重置大小功能   | Boolean           |  true       |
| rerender        | 是否开启重新渲染，不开启则调用setOptions方法，开启则调init方法   | Boolean       | false        |


::: tip 提示

查看更多属性请前往 https://www.antdv.com/components/badge-cn#API

:::
