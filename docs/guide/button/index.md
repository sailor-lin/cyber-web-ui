# g-button 按钮

### 属性

| 属性名          | 说明                 | 类型                 | 默认值                 |
| :-------------- | :------------------- | :------------------- | :------------------- |
| width         | 宽度           | String \| Number           |             |
| pattern         | 颜色模式           | "" \| "primary" \| "dark" \| "warning" \| "error" \| "gray" \| "classic"           |             |


::: tip 提示

查看更多属性请前往 https://www.antdv.com/components/button-cn#API

:::
